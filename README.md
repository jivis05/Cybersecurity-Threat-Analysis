# Cybersecurity Threat Data Analysis

## Overview

This project analyzes global cybersecurity threat data from 2015–2024 using Python, Pandas, and Matplotlib. The objective is to identify patterns in cyber attacks, understand their impact across industries and countries, and visualize trends in cybersecurity incidents over time.

The project demonstrates practical skills in data inspection, exploratory data analysis (EDA), data visualization, and cybersecurity data interpretation.

---

## Dataset

The project uses the **Global Cybersecurity Threats (2015–2024)** dataset containing 3,000 cybersecurity incidents.

The dataset includes:

- Country
- Year
- Attack Type
- Target Industry
- Financial Loss (in Million $)
- Number of Affected Users
- Attack Source
- Security Vulnerability Type
- Defense Mechanism Used
- Incident Resolution Time (in Hours)

Dataset file included:

- `Global_Cybersecurity_Threats_2015-2024.csv`

## Project Workflow

### 1. Data Inspection and Cleaning

- Loaded dataset using Pandas
- Checked dataset dimensions and structure
- Verified data types
- Checked for missing values
- Checked for duplicate records

### 2. Attack Type Analysis

- Identified the most common cyber attack categories
- Compared attack frequencies across different attack types

### 3. Country-wise Analysis

- Analyzed cybersecurity incident distribution by country
- Identified countries with the highest number of reported incidents

### 4. Industry-wise Analysis

- Examined which industries were most frequently targeted
- Compared incident counts across sectors

### 5. Financial Loss Analysis

- Calculated total financial losses associated with different attack types
- Visualized the economic impact of cyber threats

### 6. Trend Analysis

- Analyzed incident frequency over time
- Visualized yearly cybersecurity trends

---

## Key Findings

- DDoS attacks were the most common attack category in the dataset.
- Phishing and SQL Injection attacks also accounted for a significant number of incidents.
- The United Kingdom recorded the highest number of cybersecurity incidents among the countries analyzed.
- The IT industry was the most targeted sector, followed by Banking and Healthcare.
- Cyber attacks resulted in significant financial losses across all attack categories.
- Cybersecurity incidents remained persistent across multiple years, highlighting the ongoing nature of cyber threats.

## Conclusion

This project analyzed 3,000 cybersecurity incidents using Python, Pandas, and Matplotlib to identify patterns in cyber attacks, affected industries, countries, and financial impacts.

The analysis revealed that DDoS and Phishing attacks were among the most common threats, the IT sector was the most targeted industry, and cybersecurity incidents were distributed across multiple countries and years. These findings demonstrate how data analysis can be used to better understand cybersecurity trends and support informed security decision-making.

Through this project, practical skills in data cleaning, exploratory data analysis, data visualization, and cybersecurity data interpretation were developed.
